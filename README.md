## Installation

Download the repository and add the contents of the ZLColorSlider directory to your project.

## Using ZLColorSlider
```objc
-(IBAction)sliderDidChange:(RGSColorSlider *)sender{
    self.colorView.backgroundColor = sender.color;
}
```
